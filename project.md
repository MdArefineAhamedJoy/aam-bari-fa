# আমবাড়ি (AamBari) - ফ্রেশ ম্যাঙ্গো শপ

আমবাড়ি হলো রাজশাহীর প্রিমিয়াম কোয়ালিটির টাটকা এবং মিষ্টি আমের একটি অনলাইন শপ। এই প্রজেক্টের লক্ষ্য হলো গ্রাহকদের কাছে সরাসরি বাগান থেকে সংগৃহীত বিষমুক্ত আম পৌঁছে দেওয়া।

## প্রধান বৈশিষ্ট্যসমূহ (Key Features)

- **সরাসরি বাগান থেকে সংগৃহীত:** টাটকা এবং মিষ্টি আমের নিশ্চয়তা।
- **বিষমুক্ত (Chemical Free):** স্বাস্থ্যকর এবং নিরাপদ আম।
- **সুন্দর প্যাকেজিং:** নিরাপদ ডেলিভারির জন্য উন্নতমানের প্যাকেজিং।
- **সারা বাংলাদেশে হোম ডেলিভারি:** দেশের যেকোনো প্রান্তে আম পৌঁছে দেওয়া।
- **প্রিমিয়াম ইউজার ইন্টারফেস:** একটি আধুনিক এবং দৃষ্টিনন্দন ওয়েবসাইট যা ব্যবহারকারীদের কেনাকাটার অভিজ্ঞতাকে সহজ করবে।

## প্রযুক্তিগত পরিকল্পনা (Tech Stack)

- **ফ্রেমওয়ার্ক:** Next.js (App Router)
- **স্টাইলিং:** Tailwind CSS (Custom Color Palette)
- **ভাষা:** বাংলা (Bengali)
- **অ্যানিমেশন:** Framer Motion (ব্যবহারকারীর অভিজ্ঞতাকে প্রাণবন্ত করতে)
- **আইকন:** Lucide React

## ডিজাইন গাইডলাইন (Design Guidelines)

- **কালার প্যালেট:** আমের রং (হলুদ, কমলা) এবং প্রকৃতির রং (সবুজ) এর সমন্বয়ে একটি ভাইব্রেন্ট এবং ফ্রেশ লুক।
- **টাইপোগ্রাফি:** 'Hind Siliguri' বা 'Noto Sans Bengali' ব্যবহার করা হবে যা বাংলা টেক্সটকে প্রিমিয়াম লুক দেয়।
- **মোবাইল ফার্স্ট ডিজাইন:** ওয়েবসাইটটি সব ধরণের ডিভাইসে নিখুঁতভাবে কাজ করবে।
